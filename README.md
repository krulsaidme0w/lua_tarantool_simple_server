# Tarantool http server with database
---
## Installation

```sh
docker-compose -f docker-compose.yml up
```

---

## API:
 - ##### `POST     /kv       body: {key: "test", "value": {SOME ARBITRARY JSON}} `
 - ##### `PUT      kv/{id}   body: {"value": {SOME ARBITRARY JSON}}`
 - ##### `GET      kv/{id} `
 - ##### `DELETE   kv/{id}`
